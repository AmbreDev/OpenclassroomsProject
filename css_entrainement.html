Juste pour m'entrainer

Ici je note quelques infos car j'ai besoin de paragraphes pour pouvoir mettre des effets dessus et m'entrainer à changer le style en appliquants du CSS dessus

Un petit titre histoire de vérifier les différentes options

Je note en paragraphes les sélecteurs de balises spécifiques, c-a-d l'étoile qui sélectionne toutes les balises, deux noms d'affilée qui signifie la deuxième dans la première (par ex H3 em cela signifie tous les mots importants dans les titres 3, ensuite un nom + un nom que j'ai appliqué sur ce paragraphe.)
Pour d'autres voir le cours que j'ai recopié, avec le lien vers W3C

On passe aux exercices suivants

Ici il va s'agir de changer la police de mon texte, je peux choisir blah blah par exemple.
Je mets un autre paragraphe pour apprendre à changer des tailles et des caractéristiques spécifiques sur le texte, genre mettre en italique , mettre en gras , souligner une partie...

Ici un titre pour l'alignement

Je fais faire un grand paragraphe juste histoire d'avoir quelques lignes pour apprendre à justifier le texte sur plusieurs lignes. Nous arrivons maintenant à un chapitre qui devrait beaucoup vous intéresser. Non, le « formatage du texte » n'a rien à voir avec la destruction de toutes les données présentes sur votre disque dur ! Cela signifie simplement que l'on va modifier l'apparence du texte (on dit qu'on le « met en forme »). Pas de surprise particulière : nous sommes toujours dans le CSS, et nous allons réutiliser ce que nous venons d'apprendre dans le chapitre précédent. Nous allons donc travailler directement au sein du fichier .css que nous avons créé. Ce chapitre va être l'occasion de découvrir de nombreuses propriétés CSS : nous allons voir comment modifier la taille du texte, changer la police, aligner le texte…
